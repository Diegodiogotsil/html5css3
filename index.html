one page site

Ideal para negocios de pequeno ou medio porte o "one page site" é um site onde você inclui todas as informações que o vizitante precisa saber de forma objetiva e criativa em uma unica página.

Wireframe

Um wireframe é uma representação visual da estrutura básica de uma página. Ele deve ser feito da maneira mais simples possível, mostrando apenas o essencial, como um tipo de rascunho, sem cores ou imagens. Clique no botão ou escaneie o QR Code para obter o wireframe desta página.
